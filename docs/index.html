OyNIx
Nyx-Powered Local AI Browser

A custom Chromium-based desktop browser with built-in local LLM assistant, Nyx search engine, tree-style tabs, dynamic glass UI, and a 1400+ site database that grows as you browse.

Download v2.3 View Source
Features

Everything you need in a browser, with AI built in — no cloud required.

🤖

Local AI Assistant

Built-in LLM running on your CPU. Chat, summarize pages, ask questions — all offline. Supports HyperNix.1 and Nix2.6-mm models.

🔍

Nyx Search Engine

Auto-indexes every page you visit. Search your browsing history with BM25 scoring. Falls back to web results silently — never redirects.

🎨

Dynamic Glass UI

GPU-accelerated mouse-tracking refractions, glassmorphism, particle canvas, animated transitions on every page.

📂

Tree-Style Tabs

Organize tabs hierarchically in a sidebar tree. Drag, nest, collapse — manage dozens of tabs without losing context.

⚡

C++ Turbo Indexer

Multi-threaded BM25 scoring, n-gram extraction, batch document processing. 10x faster than pure Python with C# orchestration.

🔒

Password Manager

Built-in credential manager with XOR encryption, PBKDF2 key derivation, auto-fill on login pages, and passkey support.

🎵

Audio Player

Built-in audio player for media files. Play, pause, seek — right in the browser sidebar.

📦

Extension Support

Install .npi (OyNIx) and .xpi (Firefox) extensions. Content scripts, custom styles, and background actions.

📈

1400+ Site Database

Curated database that auto-expands as you browse. GitHub sync, Chrome history import, custom search engine builder.

Install

Available for Linux (Debian, Arch, Flatpak) and Windows.

🐧

Debian / Ubuntu

sudo dpkg -i oynix-2.3-amd64.deb

Requires Python 3.10+, installs PyQt6 deps automatically

🎯

Arch Linux (pacman)

sudo pacman -U oynix-2.3-1-x86_64.pkg.tar.zst

Works on X11 and Wayland. Icons included in hicolor theme.

📦

Flatpak

flatpak install --user OyNIx-2.3.flatpak

Sandboxed. KDE 6.7 runtime with Qt6 WebEngine.

🏳

Windows

OyNIx-2.3-Setup.exe

Standalone executable. No install required.

💻

From Source

pip install -r requirements.txt && python -m oynix

Clone the repo and run directly with Python 3.10+
